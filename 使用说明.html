个人链接库使用说明

这是一个本地优先的个人链接收藏和知识整理工具。适合保存 AI 工具、学习资料、GitHub 仓库、论文网站、教程页面和素材网站。

正式使用建议安装浏览器扩展。HTML 备用版已放在 html-database 文件夹。

1. 怎么安装扩展？

  1. 先解压压缩包。
  2. 打开 Chrome：chrome://extensions/，或 Edge：edge://extensions/。
  3. 打开右上角的“开发者模式”。
  4. 点击“加载已解压的扩展程序”。
  5. 选择项目里的 extension 文件夹。
  6. 把扩展图标固定到浏览器右上角。

注意：安装时不要选择整个项目根目录，要选择 extension 文件夹。

2. 怎么收藏网页？

点扩展图标收藏

弹窗会自动填标题、URL 和网页简介。你可以直接保存，也可以先修改分类、标签和笔记。

右键直接收藏

网页空白处右键可以保存当前页面；对页面里的链接右键可以保存那个链接。

3. 怎么打开完整库？

  • 点击扩展弹窗里的“打开完整库”。
  • 在网页右键菜单中选择“打开个人链接库完整库”。
  • 打开完整库后，可以把这个页面加入浏览器书签栏。

4. 完整库能做什么？

  • 按大类、小分类、标签、状态管理链接。
  • 搜索标题、网址、简介、用途和 Markdown 笔记。
  • 导入浏览器书签 HTML。
  • 导入 / 导出 JSON 备份。
  • 复制 Markdown 卡片，方便整理资料或发给别人。
  • 切换浅色 / 深色模式。

5. 数据保存在哪里？怎么备份？

数据默认保存在浏览器扩展的本地存储中，不需要账号，也不会上传到服务器。

卸载扩展、清理浏览器数据、换电脑、换浏览器之前，请先在完整库里点击“导出备份”。
