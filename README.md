# 个人链接库

这是一个双击 `index.html` 就能使用的本地个人链接库。它用于保存网站、GitHub 仓库、AI 工具、学习资料、论文资料等，并重点记录：

- 这个网站是干什么的
- 以后什么时候用它

## 怎么运行

直接双击 `index.html`。

不需要网址，不需要本地服务器，不需要数据库。

## 数据保存在哪里

默认保存到浏览器 `localStorage`。

只要你使用同一个浏览器、同一个 `index.html` 文件路径，添加和修改的数据下次打开还会在。

注意：不要在压缩包里直接打开 `index.html`。请先解压到一个固定文件夹再使用。

## JSON 自动同步

浏览器出于安全限制，网页不能偷偷写本地文件。所以如果你想让每次修改都自动同步到 JSON 文件，需要手动授权一次：

1. 打开 `index.html`
2. 点击“连接 JSON 文件”
3. 选择或创建 `links-data.json`
4. 之后添加、编辑、删除链接时，会同时保存到浏览器和这个 JSON 文件

如果不连接 JSON 文件，也可以随时点击“导出 JSON”手动备份。

## 备份和恢复

- “导出 JSON”：把当前链接库下载为备份文件
- “导入 JSON”：把备份数据合并进当前链接库，重复 URL 会自动跳过

## 当前功能

- 添加链接
- 编辑链接
- 删除链接，删除前二次确认
- 搜索标题、网址、分类、标签、简介、使用场景
- 按分类筛选
- 按状态筛选
- 按标签筛选
- 打开链接并记录最近打开时间
- JSON 导出
- JSON 导入
- 可选 JSON 文件自动同步
- 左侧筛选栏收起/展开
- 左右区域独立滚动
