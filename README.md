# Personal Link Library / 个人链接库

一个本地优先的个人链接收藏与知识整理工具。适合保存 AI 工具、学习资料、GitHub 仓库、论文网站、素材网站、教程页面和临时发现的好网站。

核心目标：**收藏快、整理清楚、搜索方便、数据掌握在自己手里**。

> 给普通用户看的图文说明：双击 `使用说明.html`。  
> GitHub 页面阅读：看本 `README.md`。


## 当前版本：v2.0.0

## 主要功能

- 浏览器扩展快速收藏当前网页
- 右键菜单收藏当前页面或页面中的某个链接
- 扩展内完整数据库工作台
- 收件箱 / 未整理工作流，适合先保存后整理
- 支持大类、小分类、标签、状态、评分、Markdown 笔记
- 自动读取网页标题、URL、简介和 favicon
- 支持导入浏览器书签 HTML
- 支持导入 / 导出 JSON 备份
- 支持复制 Markdown 卡片，方便发给别人或写 README
- 支持浅色 / 深色视觉模式
- 数据默认保存在浏览器本地扩展存储中，不依赖服务器

## 推荐使用方式

正式使用建议安装 `extension/` 文件夹里的浏览器扩展。

```text
浏览网页
↓
点扩展图标快速收藏，或右键直接收藏
↓
打开完整库整理分类、标签和笔记
↓
定期导出 JSON 备份
```

`html-database/index.html` 是本地网页版备用入口，已单独放入文件夹，避免和扩展主程序混在一起。主入口建议使用扩展版完整库。

## 快速安装

1. 解压本项目压缩包。
2. 打开 Chrome 或 Edge 的扩展管理页。
   - Chrome：`chrome://extensions/`
   - Edge：`edge://extensions/`
3. 打开右上角“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. **选择本项目里的 `extension` 文件夹**。
6. 把扩展图标固定到浏览器右上角。
7. 点击扩展图标，选择“打开完整库”。

注意：安装时不要选择整个项目根目录，要选择 `extension` 文件夹。

## 日常使用

### 快速收藏当前网页

点击扩展图标，弹窗会自动填入：

- 标题
- URL
- 网页简介
- favicon

你可以直接点“保存”，也可以先修改简介、分类、标签和笔记。

### 右键直接收藏

在网页空白处右键：

```text
保存当前页面到个人链接库
```

对页面中的链接右键：

```text
保存这个链接到个人链接库
```

右键收藏默认进入：

```text
收件箱 / 未整理
```

之后可以在完整库中统一整理分类。

### 打开完整库

可以通过以下方式打开：

- 点击扩展弹窗里的“打开完整库”
- 在网页右键菜单中选择“打开个人链接库完整库”
- 打开后把完整库页面加入浏览器书签栏

### 复制 Markdown 卡片

在完整库中，每条链接可以复制 Markdown 卡片，格式类似：

```md
[网站标题](https://example.com)

简介：这个网站用于……

用途：以后查资料时使用。

分类：AI工具 / 写作
标签：#AI #写作
状态：常用
```

## 数据和备份

数据默认保存在浏览器扩展的本地存储中。

项目不会上传你的收藏数据，也不需要账号登录。

建议定期在完整库页面点击“导出备份”，保存 JSON 文件。尤其在以下操作前必须备份：

- 卸载扩展前
- 清理浏览器数据前
- 换电脑前
- 换浏览器前
- 大量删除链接前
