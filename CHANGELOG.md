# 更新记录

## v2.0.5

- 筛选栏内容支持更完整的自由编辑。
- 分类：保留大类、小分类的新增、重命名、删除、排序和拖拽移动。
- 状态：新增“+ 状态”，支持状态新增、重命名、删除；删除状态时会把相关链接转移到其他状态。
- 标签：新增“+ 标签”，支持标签新增、重命名、删除；删除标签时会从所有链接中移除。
- 导入 / 导出 JSON 时会保留自定义状态和自定义标签。
- 快速收藏和右键收藏会优先使用当前数据库已有状态，减少固定默认状态对自定义状态的干扰。

## v2.0.4

- 所有链接行默认保持一致高度，并从列表顶部开始排列。
- 标准列表、紧凑列表、卡片视图都统一修正顶部排列效果。
